Интерактивное объяснение

В этом разделе вы найдете интерактивные примеры и объяснения, которые помогут вам лучше понять тему.

1 шаг: Повторяем ввод и вывод данных

Инструкция: Сначала давайте повторим, как выводить данные на экран с помощью print() и получать данные от пользователя с помощью input().

name = input('Введите ваше имя: ')
age = int(input('Сколько вам лет? '))
print('Привет,', name)
print('Через 5 лет вам будет', age + 5)

Пояснение: В этом примере программа сначала спрашивает имя и возраст пользователя, а потом выводит приветствие и рассчитывает возраст через 5 лет.

2 шаг: Создаём проект – персональный приветственный скрипт

Инструкция: Теперь создадим небольшой проект. Наша программа будет спрашивать имя, возраст и город пользователя, а потом выводить персональное приветствие.

name = input('Введите ваше имя: ')
age = int(input('Введите ваш возраст: '))
city = input('Введите ваш город: ')
print('Привет,', name)
print('Привет,', name + '!')
print('Ты из города', city + '.')
print('Через 10 лет тебе будет', age + 10, 'лет.')

Пояснение: Программа использует данные, которые вводит пользователь, и выводит персональное сообщение. Попробуйте изменить текст приветствия или добавить новые вопросы!

3 шаг: Сохраняем и запускаем программу

Инструкция: Теперь давайте сохраним и запустим наш проект.

  1. Выберите File → Save As..., введите имя файла, например my_project.py, и нажмите Сохранить.
  2. Для запуска программы нажмите F5 или выберите Run → Run Module.

Пояснение: После запуска вы увидите, как программа задаёт вопросы и выводит ваши ответы в виде персонализированного сообщения.



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Калькулятор покупок»

Задача:

Создайте программу в IDLE, которая будет считать стоимость покупок.

Программа должна:

  • запросить у пользователя название трёх товаров и их цену,
  • рассчитать общую сумму,
  • вывести итоговую стоимость на экран.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с вводом данных от пользователя:

    # Ввод товаров и цен
    item1 = input('Введите название первого товара: ')
    price1 = float(input('Введите цену первого товара: '))

    item2 = input('Введите название второго товара: ')
    price2 = float(input('Введите цену второго товара: '))

    item3 = input('Введите название третьего товара: ')
    price3 = float(input('Введите цену третьего товара: '))

  3. Рассчитайте общую стоимость покупок:

    # Рассчёт общей суммы
    total_price = price1 + price2 + price3

  4. Выведите итоговую стоимость на экран:

    print('')
    print('--- Ваши покупки ---')
    print(item1, '-', price1, 'тенге')
    print(item2, '-', price2, 'тенге')
    print(item3, '-', price3, 'тенге')

    print('Общая сумма:', total_price, 'тенге')
    print('Спасибо за покупку!')

Дополнительное задание

"Добавьте расчёт скидки. Программа должна спросить размер скидки и вычесть её из общей суммы."

# Ввод скидки
discount = float(input('Введите размер скидки в тенге: '))

# Итоговая сумма
final_price = total_price - discount

print('')
print('Итоговая сумма со скидкой:', final_price, 'тенге')

Проект: Персональный планировщик дня

Проект: «Персональный планировщик дня»

Задача:

Создайте программу в IDLE, которая поможет вам составить план на день.

Программа должна:

  • запросить имя пользователя,
  • спросить три запланированных дела и время их выполнения,
  • вывести готовый план на экран.

Что нужно сделать:

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE, выберите File → New File и создайте новый файл для программы."

  2. Напишите программу с вводом данных для плана дня.

    Инструкция: "Используйте input() для ввода информации о делах и print() для вывода плана на экран."

  3. Пример программы:

    # Ввод имени пользователя
    user_name = input('Введите ваше имя: ')

    # Ввод дел и времени их выполнения
    task1 = input('Введите первое дело: ')
    time1 = input('Во сколько вы начнёте это дело? ')

    task2 = input('Введите второе дело: ')
    time2 = input('Во сколько вы начнёте это дело? ')

    task3 = input('Введите третье дело: ')
    time3 = input('Во сколько вы начнёте это дело? ')

    # Вывод плана дня
    print('')
    print('--- План на день для', user_name, '---')
    print(time1, '-', task1)
    print(time2, '-', task2)
    print(time3, '-', task3)
    print('')
    print('Удачного дня,', user_name + '!')

Дополнительное задание для творческого подхода:

"Добавьте в программу приветствие в зависимости от времени дня. Например, если пользователь планирует дела утром, программа может пожелать Доброе утро!"

time_of_day = input('Утро, день или вечер? ')
print('Доброе', time_of_day + ',', user_name + '!')
