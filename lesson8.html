1 шаг: Что такое линейные алгоритмы?

Инструкция: Линейные алгоритмы — это такие программы, в которых команды выполняются строго по порядку, одна за другой. Каждая команда выполняется только один раз.

name = input('Введите ваше имя: ')
age = int(input('Сколько вам лет? '))

print('Привет,', name)
print('Через 5 лет вам будет', age + 5)

Пояснение: Программа выполняет шаги последовательно: сначала спрашивает имя, потом возраст, и в конце выводит результат. Это и есть линейный алгоритм!

2 шаг: Добавляем новые шаги в линейный алгоритм

Инструкция: Теперь давайте усложним наш алгоритм, добавив ещё несколько шагов. Программа будет спрашивать город и любимое хобби пользователя.

name = input('Введите ваше имя: ')
age = int(input('Сколько вам лет? '))
city = input('В каком городе вы живёте? ')
hobby = input('Какое у вас хобби? ')

print('Привет,', name + '!')
print('Ты из города', city)
print('Твоё хобби —', hobby)
print('Через 5 лет тебе будет', age + 5, 'лет.')

Пояснение: Мы добавили шаги для ввода города и хобби. Программа продолжает выполняться линейно — шаг за шагом, без повторений или пропусков.

3 шаг: Сохраняем и запускаем линейный алгоритм

Инструкция: Чтобы сохранить и запустить программу:

  1. Выберите File → Save As..., введите имя файла, например linear_algorithm.py, и нажмите Сохранить.
  2. Для запуска программы нажмите F5 или выберите Run → Run Module.

Пояснение: После запуска программа задаст вопросы и выведет ответы в нужном порядке. Вы создали свой первый линейный алгоритм!



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Планировщик дня»

Задача:

Создайте программу в IDLE, которая будет планировать ваш день.

Программа должна:

  • запросить у пользователя время начала дня,
  • спросить три запланированных дела и их время,
  • вывести готовый план на экран.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с вводом данных от пользователя:

    # Ввод времени начала дня
    start_time = input('Во сколько начинается ваш день? ')

    # Ввод дел и их времени
    task1 = input('Введите первое дело: ')
    time1 = input('Во сколько вы его начнёте? ')

    task2 = input('Введите второе дело: ')
    time2 = input('Во сколько вы его начнёте? ')

    task3 = input('Введите третье дело: ')
    time3 = input('Во сколько вы его начнёте? ')

  3. Выведите план на экран:

    print('')
    print('--- План на день ---')
    print('Начало дня в', start_time)
    print(time1, '-', task1)
    print(time2, '-', task2)
    print(time3, '-', task3)
    print('')
    print('Удачного дня!')

Дополнительное задание

"Добавьте в программу расчёт времени до первого дела. Программа должна спросить текущее время и рассчитать, сколько времени осталось до начала первого дела."

# Ввод текущего времени
current_time = int(input('Введите текущее время (только часы): '))
first_task_time = int(time1)

# Рассчёт оставшегося времени
time_left = first_task_time - current_time

print('')
print('До первого дела осталось', time_left, 'часов.')
