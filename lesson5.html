1 шаг: Основные арифметические операции

Инструкция: В Python можно выполнять разные арифметические операции. Вот основные из них:

  • + — сложение
  • - — вычитание
  • * — умножение
  • / — деление
  • // — целочисленное деление
  • % — остаток от деления
  • ** — возведение в степень
a = 10
b = 3
print = (a + b)ㅤㅤ# Сложение: 13
print = (a - b)ㅤㅤ# Вычитание: 7
print = (a * b)ㅤㅤ# Умножение: 30
print = (a / b)ㅤㅤ# Деление: 3.333...

Пояснение: Вы можете выполнять простые вычисления прямо в коде. Обратите внимание: результат деления всегда число с точкой (тип float).

2 шаг: Специальные арифметические операции

Инструкция: В Python есть и более сложные арифметические операции. Давайте посмотрим, как они работают.

# Целочисленное деление
print = (a // b)ㅤㅤ# Результат: 3
# Остаток от деления
print = (a % b)ㅤㅤ# Результат: 1
# Возведение в степень
print = (a ** b)ㅤㅤ# Результат: 1000

Пояснение: Целочисленное деление (//) убирает дробную часть и оставляет только целое число. Операция % показывает остаток от деления. Возведение в степень (**) работает так: a в степени b.

3 шаг: Порядок выполнения операций

Инструкция: Python выполняет арифметические операции в определённом порядке. Сначала умножение и деление, потом сложение и вычитание. Если нужно изменить порядок, используйте скобки.

result = 2 + 3 * 4
print = (result)ㅤㅤ# Результат: 14 (сначала умножение)
# Изменим порядок с помощью скобок
result = (2 + 3) * 4
print = (result)ㅤㅤ# Результат: 20 (сначала сложение)

Пояснение: Без скобок Python сначала умножает, а потом складывает. Со скобками сначала выполняется то, что внутри них.



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Калькулятор для школы»

Задача:

Создайте программу в IDLE, которая будет выполнять простые арифметические операции.

Программа должна:

  • запросить у пользователя два числа,
  • выполнить с ними сложение, вычитание, умножение и деление,
  • показать результат на экране.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с вводом двух чисел:

    # Ввод двух чисел от пользователя
    number1 = int(input('Введите первое число: '))
    number2 = int(input('Введите второе число: '))

  3. Выполните арифметические операции:

    # Сложение
    sum_result = number1 + number2
    print('Сумма:', sum_result)

    # Вычитание
    difference = number1 - number2
    print('Разность:', difference)

    # Умножение
    product = number1 * number2
    print('Произведение:', product)

    # Деление
    division = number1 / number2
    print('Частное:', division)

Дополнительное задание

"Добавьте в программу операции целочисленного деления и остатка от деления."

# Целочисленное деление
integer_division = number1 // number2
print('Целочисленное деление:', integer_division)

# Остаток от деления
remainder = number1 % number2
print('Остаток от деления:', remainder)


Самостоятельное задание

Попробуйте применить изученный материал, создав свою собственную программу или решение.

Проект: Личный калькулятор

Проект: «Личный калькулятор»

Задача:

Создайте программу в IDLE, которая будет выполнять арифметические вычисления.

Программа должна:

  • запросить у пользователя два числа,
  • выполнить с ними основные арифметические операции (сложение, вычитание, умножение, деление),
  • вывести результаты на экран.

Что нужно сделать:

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE, выберите File → New File и создайте новый файл для программы."

  2. Напишите программу с вводом двух чисел от пользователя.

    Инструкция: "Используйте input() для ввода чисел и преобразуйте их в целые числа с помощью int()."

  3. Пример программы:

    # Ввод чисел от пользователя
    number1 = int(input('Введите первое число: '))
    number2 = int(input('Введите второе число: '))

    # Выполнение арифметических операций
    sum_result = number1 + number2
    difference = number1 - number2
    product = number1 * number2
    division = number1 / number2

    # Вывод результатов
    print('Сумма:', sum_result)
    print('Разность:', difference)
    print('Произведение:', product)
    print('Частное:', division)

  4. Сохраните и запустите программу (нажмите F5).

    Инструкция: "Сохраните программу через File → Save As..., назовите файл, например calculator.py, и нажмите Сохранить."

    Нажмите F5 для запуска программы и введите два числа.

Дополнительное задание для творческого подхода:

"Добавьте операции целочисленного деления и нахождения остатка от деления."

# Целочисленное деление и остаток от деления
integer_division = number1 // number2
remainder = number1 % number2

print('Целочисленное деление:', integer_division)
print('Остаток от деления:', remainder)
