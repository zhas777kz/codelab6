Интерактивное объяснение

В этом разделе вы найдете интерактивные примеры и объяснения, которые помогут вам лучше понять тему.

1 шаг: Повторяем, что такое линейные алгоритмы

Инструкция: Напомним, линейные алгоритмы — это программы, которые выполняются по шагам, один за другим, без повторений и ветвлений.

name = input('Введите ваше имя: ')
age = int(input('Сколько вам лет? '))
print('Привет,', name)
print('Через 5 лет тебе будет', age + 5)

Пояснение: Каждая команда выполняется строго по порядку. Это и есть основа линейных алгоритмов.

2 шаг: Создаём проект – личная визитка

Инструкция: Теперь давайте создадим проект — программу, которая будет работать как личная визитка. Она будет спрашивать имя, возраст, город и хобби, а потом выводить готовое сообщение.

name = input('Введите ваше имя: ')
age = int(input('Сколько вам лет? '))
city = input('Где вы живёте? ')
hobby = input('Какое у вас хобби? ')

print('--- Ваша визитка ---')
print('Имя:', name)
print('Возраст:', age)
print('Город:', city)
print('Хобби:', hobby)

Пояснение: Программа последовательно собирает информацию и выводит её в виде визитки. Вы можете изменить текст или добавить новые поля.

3 шаг: Сохраняем и запускаем проект

Инструкция: Чтобы сохранить и запустить проект:

  1. Выберите File → Save As..., введите имя файла, например my_card.py, и нажмите Сохранить.
  2. Для запуска программы нажмите F5 или выберите Run → Run Module.

Пояснение: После запуска программа задаст вопросы и создаст персональную визитку. Поздравляем, вы создали свой проект с линейным алгоритмом!



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Персональный дневник настроения»

Задача:

Создайте программу в IDLE, которая будет работать как персональный дневник настроения.

Программа должна:

  • спросить имя пользователя и дату,
  • запросить три вопроса о настроении и активности,
  • вывести короткое сообщение с итогами дня.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с вводом данных от пользователя:

    # Ввод имени и даты
    name = input('Введите ваше имя: ')
    date = input('Какое сегодня число? ')

    # Ввод данных о настроении
    mood = input('Какое у вас сегодня настроение? ')
    activity = input('Чем вы сегодня занимались? ')
    goal = input('Удалось ли вам достичь своих целей сегодня? ')

  3. Выведите итоговое сообщение на экран:

    print('')
    print('--- Дневник настроения ---')
    print('Имя:', name)
    print('Дата:', date)
    print('Настроение:', mood)
    print('Сегодня вы занимались:', activity)
    print('Цели достигнуты:', goal)
    print('')
    print('Спасибо, что поделились,', name + '! Удачи завтра!')

Дополнительное задание

"Добавьте возможность сохранить результаты в файл. Программа должна записывать ответы в текстовый файл с названием mood_diary.txt."

# Сохранение данных в файл
with open('mood_diary.txt', 'w') as file:
    file.write('--- Дневник настроения ---\n')
    file.write('Имя: ' + name + '\n')
    file.write('Дата: ' + date + '\n')
    file.write('Настроение: ' + mood + '\n')
    file.write('Сегодня вы занимались: ' + activity + '\n')
    file.write('Цели достигнуты: ' + goal + '\n')

print('Ваш дневник сохранён в файл mood_diary.txt!')
