1 шаг: Что такое алфавит языка?

Инструкция: Язык программирования Python использует специальный алфавит — это набор символов, с помощью которых пишется код. В этот алфавит входят:

  • Английские буквы (большие и маленькие)
  • Цифры
  • Специальные символы, такие как +, -, =, :, и другие
name = 'Alina'   # Английские буквы
age = 12         # Цифры
print(name, age)

Пояснение: В этом примере мы использовали буквы для имени переменной и цифры для возраста. Помните, что Python различает большие и маленькие буквы!

2 шаг: Основы синтаксиса в Python

Инструкция: Синтаксис — это правила, по которым пишутся программы. В Python важно соблюдать правильное оформление кода:

  • Использовать двоеточие (:) после команд, которые требуют блока кода.
  • Обязательно следить за отступами (4 пробела или клавиша Tab).

Код для демонстрации (правильный):

name = 'Alina'
print('Привет,', name)

Код с ошибкой (неправильный):

name = 'Alina'
print('Привет,' name)  # Ошибка: пропущена запятая

Пояснение: Если забыть запятую или кавычки, программа покажет ошибку. Всегда проверяйте правильность написания команд.

3 шаг: Создаём свою первую программу с синтаксисом

Инструкция: Теперь давайте создадим программу, которая расскажет немного о вас. В этой программе мы будем использовать переменные и выводить текст на экран.

name = 'Alina'
hobby = 'drawing'
age = 12

print('Меня зовут', name)
print('Мне', age, 'лет')
print('Моё хобби —', hobby)

Пояснение: Мы создали три переменные: name, hobby и age, и вывели их на экран с помощью команды print(). Попробуйте изменить значения переменных и посмотреть, как изменится результат.



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Проверка синтаксиса»

Задача:

Создайте программу в IDLE, которая будет рассказывать о вас, но с правильным синтаксисом! Ваша программа должна содержать:

  • имя,
  • возраст,
  • любимый цвет.

Проверьте, чтобы программа не выдавала ошибок!

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с правильным синтаксисом:

    name = 'Alina'
    age = 12
    favorite_color = 'blue'

    print('Меня зовут', name)
    print('Мне', age, 'лет')
    print('Мой любимый цвет —', favorite_color)

  3. Сохраните программу и запустите её (нажмите F5).

    Инструкция: "Сохраните программу через File → Save As... и нажмите F5 для запуска. Если программа показывает ошибки — проверьте кавычки, запятые и правильное написание переменных."

Подсказки и примеры

  • Подсказка 1: "Убедитесь, что текст находится в кавычках ('текст')."
  • Подсказка 2: "Переменные должны быть названы правильно:
    • использовать только английские буквы,
    • нельзя использовать пробелы (заменяйте их нижним подчёркиванием _).

Дополнительное задание

"Добавьте запрос на ввод данных от пользователя. Программа должна спросить имя и любимый цвет, а затем вывести их."

user_name = input('Как вас зовут? ')
favorite_color = input('Ваш любимый цвет: ')

print('Приятно познакомиться,', user_name + '!')
print('Твой любимый цвет —', favorite_color)
