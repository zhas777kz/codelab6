Интерактивное объяснение

1 шаг: Знакомство с типами данных

Инструкция: В Python данные могут быть разных типов. Основные типы данных:

  • int — целые числа (integer)
  • float — числа с десятичной точкой (floating point)
  • str — строки, то есть текст (string)
  • bool — логические значения: True (истина) и False (ложь)
age = 12ㅤㅤㅤㅤㅤㅤㅤ # int (целое число)
height = 1.65ㅤㅤㅤㅤ # float (число с точкой)
name = 'Alina'ㅤ ㅤㅤ # str (строка)
is_student = Trueㅤㅤ# bool (логическое значение)

Пояснение: Мы создали переменные с разными типами данных. age — это целое число, height — число с точкой, name — текст, а is_student — логическое значение.

2 шаг: Проверка типов данных

Инструкция: Иногда нужно узнать, какого типа данные находятся в переменной. Для этого используется команда type().

print = (type(age))
print = (type(height))
print = (type(name))
print = (type(is_student))

Пояснение: Команда type() показывает тип данных, хранящихся в переменной. Попробуйте проверить тип данных для других переменных.

3 шаг: Работа с разными типами данных

Инструкция: В Python можно выполнять операции с разными типами данных, но важно следить за тем, чтобы они были совместимы. Например, нельзя просто так сложить число и строку.

Код с ошибкой:

age = 12
print = ('Мне' + age + 'лет')ㅤㅤ# Ошибка!

Правильный код:

age = 12
print = ('Мне', age, 'лет')ㅤㅤ# Работает правильно

Пояснение: Чтобы объединить текст и число, нужно использовать запятые в команде print(). Это позволит Python правильно обработать данные разных типов.



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Угадай тип данных!»

Задача:

Создайте программу в IDLE, которая создаёт переменные разных типов данных и выводит их на экран.

Ваша программа должна содержать:

  • целое число (int),
  • число с точкой (float),
  • текст (str),
  • логическое значение (bool).

В конце программа должна показать тип каждого значения.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с разными типами данных:

    # Создаём переменные разных типов
    age = 12 # int (целое число)
    height = 1.65 # float (число с точкой)
    name = 'Alina' # str (строка)
    is_student = True # bool (логическое значение)

    print('Имя:', name)
    print('Возраст:', age)
    print('Рост:', height)
    print('Студент:', is_student)

  3. Добавьте проверку типа данных с помощью type():

    print('Тип переменной name:', type(name))
    print('Тип переменной age:', type(age))
    print('Тип переменной height:', type(height))
    print('Тип переменной is_student:', type(is_student))

  4. Сохраните программу и запустите её (нажмите F5).

    Инструкция: "Сохраните программу через File → Save As... и нажмите F5 для запуска. Убедитесь, что все данные и их типы отображаются правильно."

Дополнительное задание

"Добавьте ввод данных от пользователя, а затем определите их тип."

user_age = input('Сколько вам лет? ')
print('Ваш возраст:', user_age)
print('Тип переменной user_age:', type(user_age))

# Преобразуем введённый возраст в число
user_age = int(user_age)
print('Возраст через 5 лет:', user_age + 5)
print('Новый тип переменной user_age:', type(user_age))

Проект: «Опиши себя с помощью типов данных»

Задача:

Создайте программу в IDLE, которая будет использовать разные типы данных для описания вас. Программа должна содержать:

  • текстовые данные (str),
  • числовые данные (int и float),
  • логическое значение (bool).

Выведите всю информацию на экран.

Что нужно сделать:

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE, выберите File → New File и создайте новый файл для программы."

  2. Напишите программу с использованием разных типов данных.

    # Текстовые данные (строка)

    Нажмите F5 для запуска программы и посмотрите, как программа выводит информацию.
