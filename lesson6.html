Интерактивное объяснение

В этом разделе вы найдете интерактивные примеры и объяснения, которые помогут вам лучше понять тему.

1 шаг: Как выводить данные на экран?

Инструкция: В Python для вывода данных на экран используется команда print(). С её помощью можно показывать текст и значения переменных.

print('Привет, мир!')

Пояснение: Введите эту команду в Shell и нажмите Enter. На экране появится текст "Привет, мир!". Так программа выводит данные на экран.

2 шаг: Вывод текста и переменных

Инструкция: С помощью print() можно выводить не только текст, но и значения переменных. Переменные пишутся без кавычек.

name = 'Alina'
age = 12

print('Меня зовут', name)
print('Мне', age, 'лет')

Пояснение: Здесь мы вывели текст и переменные name и age. Python автоматически добавляет пробелы между элементами при выводе.

3 шаг: Как получить данные от пользователя?

Инструкция: Чтобы программа спрашивала данные у пользователя, используем команду input(). Пользователь вводит данные, а программа сохраняет их в переменную.

name = input('Введите ваше имя: ')
print('Привет,', name)

Пояснение: Команда input() выводит сообщение и ждёт, пока пользователь введёт данные. Всё, что вводится, сохраняется в переменной name.

4 шаг: Преобразование ввода в числа

Инструкция: По умолчанию данные из input() сохраняются как текст. Если нужно ввести число, используйте int() или float() для преобразования.

age = int(input('Сколько тебе лет? '))
print('Через 5 лет тебе будет', age + 5)

Пояснение: Здесь мы преобразовали введённое значение в число с помощью int(). Теперь программа может использовать его для вычислений.

Мини-игра: «Создай своё расписание»

Задача:

Создайте программу в IDLE, которая будет составлять расписание на день.

Программа должна:

  • запросить у пользователя имя,
  • спросить три любимых школьных предмета и время их начала,
  • вывести готовое расписание на экран.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с вводом данных от пользователя:

    # Ввод имени пользователя
    name = input('Введите ваше имя: ')

    # Ввод расписания
    subject1 = input('Введите ваш первый любимый предмет: ')
    time1 = input('Во сколько начинается этот предмет? ')

    subject2 = input('Введите ваш второй любимый предмет: ')
    time2 = input('Во сколько начинается этот предмет? ')

    subject3 = input('Введите ваш третий любимый предмет: ')
    time3 = input('Во сколько начинается этот предмет? ')

  3. Выведите расписание на экран:

    print('')
    print('--- Расписание для', name, '---')
    print(time1, '-', subject1)
    print(time2, '-', subject2)
    print(time3, '-', subject3)
    print('')
    print('Желаем удачного дня,', name + '!')

Дополнительное задание

"Добавьте возможность изменить расписание. После ввода программа должна спросить: Хотите изменить расписание? Если пользователь введёт да, программа должна позволить снова ввести данные."

change_schedule = input('Хотите изменить расписание? (да/нет): ')

if change_schedule.lower() == 'да':
    subject1 = input('Введите новый первый предмет: ')
    time1 = input('Новое время для первого предмета: ')
    print('')
    print('--- Обновлённое расписание ---')
    print(time1, '-', subject1)
