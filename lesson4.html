Интерактивное объяснение

1 шаг: Основные группы типов данных

Инструкция: Все данные в Python можно разделить на несколько групп:

  • Числовые типы данных — int (целые числа), float (дробные числа).
  • Текстовые данные — str (строки).
  • Логические данные — bool (значения True или False).
# Числовые данные
age = 12
height = 1.65

# Текстовые данные
name = 'Alina'

# Логические данные
is_student = True

Пояснение: Мы создали переменные для числовых, текстовых и логических данных. Каждая переменная имеет свой тип данных.

2 шаг: Как узнать тип данных?

Инструкция: Иногда нужно проверить, какой тип данных хранится в переменной. Для этого используем команду type().

print(type(age))        # 
print(type(height))     # 
print(type(name))       # 
print(type(is_student)) #

Пояснение: Команда type() показывает тип данных переменной. Это помогает понять, как правильно использовать данные в программе.

3 шаг: Преобразование типов данных

Инструкция: Иногда нужно преобразовать один тип данных в другой. Например, если пользователь вводит число, оно автоматически сохраняется как строка. Чтобы использовать его как число, нужно преобразовать в int.

# Ввод числа от пользователя
age_input = input('Сколько тебе лет? ')
print(type(age_input))  # 

# Преобразование строки в число
age = int(age_input)
print(type(age))        #

Пояснение: Когда вы вводите данные через input(), они сохраняются как строка (str). Чтобы использовать их в арифметических операциях, нужно преобразовать в число с помощью int().

Мини-игра: «Определи свой тип!»

Задача:

Создайте программу в IDLE, которая будет принимать данные от пользователя, а затем определять и показывать их тип.

Программа должна:

  • запросить у пользователя имя, возраст и рост,
  • вывести все данные на экран,
  • определить тип каждой переменной с помощью type().

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File."

  2. Напишите программу с вводом данных от пользователя:

    # Ввод данных от пользователя
    name = input('Введите ваше имя: ')
    age = input('Сколько вам лет? ')
    height = input('Введите ваш рост (в метрах): ')

    print('Ваше имя:', name)
    print('Ваш возраст:', age)
    print('Ваш рост:', height)

  3. Определите тип данных каждой переменной:

    print('Тип переменной name:', type(name))
    print('Тип переменной age:', type(age))
    print('Тип переменной height:', type(height))

  4. Преобразуйте строки в числа:

    # Преобразуем возраст и рост в числа
    age = int(age)
    height = float(height)

    print('Новый тип переменной age:', type(age))
    print('Новый тип переменной height:', type(height))

Дополнительное задание

"Добавьте новый ввод — логический вопрос. Программа должна спрашивать, являетесь ли вы студентом, и определять тип ответа."

is_student = input('Вы студент? (да/нет): ')

# Проверим, является ли пользователь студентом
if is_student.lower() == 'да':
    is_student = True
else:
    is_student = False

print('Студент:', is_student)
print('Тип переменной is_student:', type(is_student))
