урок 1: Знакомство с IDLE

Введение в тему

Этот урок поможет вам изучить основные понятия и подготовиться к выполнению заданий.



Интерактивное объяснение

В этом разделе вы найдете интерактивные примеры и объяснения, которые помогут вам лучше понять тему.

1 шаг: Открываем IDLE и знакомимся с интерфейсом

Инструкция: Откройте программу IDLE. Перед вами появится окно Shell — это место, где можно вводить команды и сразу видеть результат.

print('Привет, мир!')

Пояснение: Введите эту команду в Shell и нажмите Enter. Вы увидите, как текст 'Привет, мир!' появится на экране. Это ваша первая команда в Python!

2 шаг: Создание нового файла для программы

Инструкция: Чтобы написать более сложную программу, создадим новый файл. В верхнем меню выберите File → New File.

name = 'Alina'
hobby = 'drawing'

print('Меня зовут', name)
print('Моё хобби —', hobby)

Пояснение: Эта программа создаёт две переменные name и hobby, а затем выводит информацию о вас.

3 шаг: Запуск программы в IDLE

Инструкция: Теперь давайте запустим программу. Сначала сохраните файл: выберите File → Save, придумайте имя файла (например, my_program.py) и нажмите Сохранить.

Чтобы запустить программу, нажмите F5 или выберите Run → Run Module.

Пояснение: В Shell появится результат:

Меня зовут Alina
Моё хобби — drawing

Поздравляем! Вы только что создали и запустили свою первую программу.



Практическое задание

Попробуйте выполнить задания, чтобы закрепить полученные знания на практике.

Мини-игра: «Создай свою визитку»

Задача:

Создайте программу в IDLE, которая будет вашей личной визиткой! Программа должна выводить:

  • ваше имя,
  • возраст,
  • хобби,
  • любимый предмет в школе.

В конце программа должна попрощаться с пользователем.

Пошаговые инструкции для выполнения задания

  1. Создайте новый файл в IDLE.

    Инструкция: "Откройте IDLE и выберите File → New File для создания новой программы."

  2. Напишите программу, используя print():

    print('Привет! Меня зовут Alina.')
    print('Мне 12 лет.')
    print('Моё хобби — рисование.')
    print('Мой любимый предмет — математика.')
    print('')
    print('Спасибо за внимание! До встречи!')

  3. Сохраните программу и запустите её (нажмите F5).

    Инструкция: "Сохраните программу через File → Save As... Убедитесь, что файл сохранён с расширением .py. После этого нажмите F5 для запуска программы.

    Проверьте:

    • Правильно ли выводится информация?
    • Если есть ошибки — проверьте кавычки и синтаксис, исправьте их.

Дополнительное задание

"Добавьте в свою визитку запрос на ввод имени пользователя с помощью команды input(). Программа должна приветствовать человека, который её запустит!"

user_name = input('Как вас зовут? ')
